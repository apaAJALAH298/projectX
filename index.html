FAQ

Web ini maksudnya buat apa sih?

Ini portfolio public R.M Erlangga — tempat kumpulin project, skill yang lagi dipelajari, dan cara paling gampang buat dihubungi di web ini jika tidak on di tiktok dan WA.

Ini dibikin pakai apa?

Cuma HTML, CSS, dan JavaScript polos — nggak pakai framework, jadi ringan dan gampang dikustomisasi dan ditambah sesuatu biar bagus dan menarik gitulah.

Robotnya beneran AI?

Belum — jawabannya udah disiapin (scripted), bukan nyambung ke model AI beneran. Tapi tetep seru buat diajak "ngobrol" santai soalnya blom ambil key ainya wkwkw.

Boleh dicontek buat portfolio sendiri?

Boleh banget, tinggal ganti nama, foto, project, dan link kontaknya jadi punya kamu dan juga ini FREE eitss tapi wajib ada vs code ya.
